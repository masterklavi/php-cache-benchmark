# PHP Cache Benchmark
Calculation and Comparison of RPS for some popular User Caches in PHP.

# List of User Caches
* File
* MySQL
* SQLite
* Shared Memory
* APC
* Memcache
* Memcached
* Redis (phpredis)

# Run
To Benchmarking:
    php run.php
To Charting the results:
    php chart.php
